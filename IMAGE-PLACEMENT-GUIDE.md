## Notes
- Recommended image sizes:
  - Hero images: 1920×1080px minimum
  - Cover/portfolio images: 1200×800px minimum
  - Architect photo: 800×1067px minimum (3:4 portrait ratio)
- Use descriptive filenames exactly as listed above (the code references them)
- WebP format is preferred but JPG works fine
